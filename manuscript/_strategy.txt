二、调整后的故事线
核心论点: 在急性心肌 I/R 损伤中，Ogt 下调通过 RELA/NF-κB 间接激活 Nrf2，髓系细胞通过 SPP1-CD44 旁分泌放大铁死亡信号，铁蛋白自噬释放游离铁驱动心肌细胞铁死亡。该机制是急性损伤阶段特异性的，在慢性心衰中不保守。
① 触发: Ogt 下调（log₂FC = −0.56，padj = 3×10⁻⁷），OGT 底物网络广泛扰动（382/6556 DEGs）
② 中介: Nrf2/铁死亡核心不是 OGT 直接底物 → RELA（NF-κB p65）是唯一上游信号连接点（log₂FC = +1.13）
③ 空间格局: snRNA-seq：Ogt 在心肌/周细胞，Nrf2 靶基因在髓系 → 旁分泌假说
④ 旁分泌验证: SPP1-CD44（CellChat + CellPhoneDB 双方法确认），CD44 是 Nrf2 靶基因
⑤ 翻译层面: mRNA-蛋白 R=0.253，50.7% 方向一致。FTH1 蛋白↓（铁蛋白自噬）、TFRC 蛋白维持（铁输入）、HMOX1/SLC7A11 蛋白低于 MS 检测限
⑥ 疾病特异性: GTEx 健康心脏：Ogt 高、Nrf2 靶基因沉默。GSE57338 慢性心衰：Nrf2 靶基因反向表达 → 急性 I/R 特异性
三、四项验证结果汇总
Step ③ OGT 底物靶向富集
数据: OGT-PIN (782 人类高置信度 OGT 互作蛋白) × GSE193997 DEGs (6,556, padj<0.05)
结果: 382 个重叠基因 (5.83%), Fisher OR=2.02, P<0.000001, 280 UP / 102 DOWN
关键: NFE2L2, KEAP1, BACH1, HMOX1, SLC7A11, GPX4 均不在 OGT-PIN 中 → OGT 对 Nrf2/铁死亡的调控是间接的
信号中介: RELA/NF-κB p65 是唯一同时属于 OGT 互作组且显著上调的转录因子 (log₂FC=+1.13, padj=5.3×10⁻¹⁵)
Step ④ 外部独立验证
GTEx v8 (n=431 健康左心室): OGT 39.2 TPM, HMOX1 3.0 TPM, SLC7A11 ~0 TPM → 健康开关模式
GSE57338 (177 HF vs 136 NF, 人慢性心衰): 方向一致性 31.5% (低于随机), Nrf2 靶基因在慢性心衰中方向反转
→ 该信号是急性 I/R 阶段特异性的，非泛化心衰标志物
Step ① 细胞通讯双方法交叉验证
方法: CellChat ∩ CellPhoneDB (LIANA 框架), 9 种细胞类型, GSE240848 snRNA-seq
共识: 8 对交互 (髓系→心肌/周细胞), SPP1-CD44 为核心信号 (巨噬+中性粒→心肌+周细胞)
关联: CD44 是 Nrf2 靶基因，介导铁内吞 → 连接旁分泌与铁死亡
Step ② mRNA-蛋白偏离分析（残差法）
数据: PXD046631 (2,308 个匹配基因), 线性回归: R=0.253, 斜率=0.144
方向一致: 50.7% (1171/2308), 方向背离: 49.3%
铁死亡关键: FTH1 背离 (mRNA↑蛋白↓ → 铁蛋白自噬), TFRC 背离 (mRNA↓蛋白→ → 铁输入维持)
Nrf2 靶基因 HMOX1, SLC7A11 蛋白低于 MS 检测限 (mRNA 强诱导但蛋白未检出 → 翻译延迟或降解)

七、论文一写作指南
7.1 Cell Reports 投稿要求
字数 ≤7,000 (不含STAR Methods、参考文献、图注)
图表 ≤7个主图/表
结构: Title -> Summary (<=150 words) -> Introduction -> Results -> Discussion -> STAR Methods
STAR Methods: Resource Availability / Experimental Model / Method Details / Quantification
所有组学数据必须存入公共数据库(GEO, PRIDE/ProteomeXchange)，文中声明accession number
7.2 叙事框架
多组学论文最致命的错误是按技术平台逐个报告结果，变成数据列表。核心原则：按生物学发现组织叙事，而非按技术平台。
核心一句话 (North Star): 急性心肌I/R中，Ogt下调通过RELA/NF-kB间接激活Nrf2，髓系SPP1-CD44旁分泌放大铁死亡信号，铁蛋白自噬加剧心肌损伤。该机制是急性损伤阶段特异性的。
7.3 推荐的Results子标题 (非技术平台命名)
1. Multi-omics profiling identifies Ogt downregulation and Nrf2/ferroptosis co-activation in acute cardiac I/R [Fig 1-2]
2. Cross-platform proteomics reveals predominant post-transcriptional regulation and ferroptosis protein-level signatures [Fig 3]
3. Single-nucleus transcriptomics uncovers cell-type dichotomy: Ogt in cardiomyocytes, Nrf2 effectors in myeloid cells [Fig 4]
4. Dual-method CCC inference confirms SPP1-CD44 as a myeloid-derived paracrine ferroptosis amplifier [Fig 5]
5. OGT substrate network perturbation reveals RELA/NF-kB as the signaling intermediary to Nrf2 [Fig 6]
6. External validation confirms acute I/R specificity [Fig 7]
7.4 Discussion 五段模型
Para 1 - 核心发现总结: 简洁重述Ogt->RELA->Nrf2->铁死亡轴及其急性I/R特异性
Para 2 - 机制解读: OGT底物不是Nrf2核心 -> RELA是中介 -> O-GlcNAc对NF-kB的调控 -> 转录后调控层
Para 3 - 旁分泌放大模型: SPP1-CD44双方法确认 -> CD44是Nrf2靶基因 -> 铁内吞 -> 铁死亡敏感性
Para 4 - 疾病阶段特异性: GTEx健康开关 + GSE57338慢性心衰反转 -> 急性损伤反应 vs 慢性重构 -> 时间窗口治疗意义
Para 5 - 局限性与展望: 小鼠模型需验证; 蛋白组覆盖度有限; 布尔网络为后续机制验证提供路线图(引用论文二)

7.6 常见写作陷阱
X Figure 2A shows the volcano plot... -> 正确: Ogt was significantly downregulated (log2FC=-0.56, padj=3e-7), accompanied by Nrf2 target gene induction (Fig 2A).
X Next, we performed proteomics analysis... -> 正确: To determine whether transcriptional changes propagated to the protein level...
X 用Interestingly/Surprisingly等词 -> 去掉，让读者自己判断
X 在Results里写Discussion式推测 -> 留给Discussion
X 按技术平台组织子标题 -> 按生物学发现组织
X 报告所有6,777 DEGs细节 -> 聚焦核心通路，细节放Supplementary
X 忽略阴性结果 -> GSE57338中Nrf2靶基因反向变化应主动报告，这是亮点
X 每段以Figure序号开头 -> 每段以科学问题/假设开头
