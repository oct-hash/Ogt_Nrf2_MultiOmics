Experimental Model and Subject Details
Mouse Model of Myocardial Ischemia-Reperfusion
The primary discovery dataset (GSE193997) profiled left ventricular tissue from wild-type C57BL/6 mice subjected to 45 min of left anterior descending (LAD) coronary artery ligation followed by reperfusion for 0.5, 1.0, 1.5, 3.0, 6.0, or 12.0 hours, with sham-operated controls (n = 3 per group; 24 samples total). Details of mouse age, sex, and anesthesia were as described in the original publication (Liu et al., 2022, PMID: 35843476). The independent validation cohort (GSE307385) profiled ischemic-region left ventricular tissue from mice subjected to 30 min of LAD ligation followed by 0, 1, 6, or 24 h reperfusion, with sham controls (n = 3 per group; 15 samples total; Zhang et al., 2025). The snRNA-seq dataset (GSE240848) was generated from rat (Rattus norvegicus) myocardial I/R tissue; full experimental details are provided in the original GEO deposit. For proteomics, PXD046631 profiled mouse left ventricular tissue after 30 min ischemia + 6 h reperfusion vs. sham (LFQ, n = 4 per group). PXD040135 profiled mouse cardiac tissue after 1 h LAD ligation + 24 h reperfusion vs. sham (LFQ, n = 3 per group). PXD035154 profiled mouse cardiac tissue at 2-day and 2-week post-I/R time points (TMT10plex; Pool 1: 2-day I/R, Sham n = 2, I/R n = 3; Pool 2: 2-week I/R, Sham n = 2, I/R n = 3; Hofmann et al., 2024).
Human Heart Failure Samples (GSE57338)
The external validation dataset GSE57338 consists of Affymetrix Human Gene 1.0 ST Array (GPL11532) expression profiles from left ventricular myocardial samples of 313 individuals, including 177 with heart failure (idiopathic dilated cardiomyopathy or ischemic cardiomyopathy) and 136 non-failing controls. All samples were obtained with informed consent and institutional review board approval as described in the original publication (Liu et al., 2015, PMID: 25528681).
GTEx v8 Healthy Controls
Median gene-level TPM values for Heart - Left Ventricle (n = 431) were obtained from the Genotype-Tissue Expression (GTEx) project v8 release via the GTEx Portal API v2 (https://gtexportal.org). GTEx donor characteristics and tissue procurement procedures are described in the GTEx consortium publications.
Method Details
Bulk RNA-seq Processing and Differential Expression Analysis
Raw RNA-seq reads from GSE193997 and GSE307385 were aligned to the mouse reference genome (GRCm38/mm10) using STAR v2.7. Gene-level counts were quantified with featureCounts. Differential expression analysis was performed using DESeq2 (v1.34). Genes with adjusted P-value (Benjamini-Hochberg) < 0.05 were considered differentially expressed. For the primary discovery analysis in GSE193997 (I/R 6h vs Sham), 6,556 DEGs met this threshold. The two bulk RNA-seq datasets were not merged but analyzed as independent cohorts with distinct roles. GSE193997 (45 min ischemia) served as the discovery cohort for DEG analysis and WGCNA, while GSE307385 (30 min ischemia) provided independent time-series validation of Ogt/Oga expression dynamics and external replication of WGCNA module eigengenes. Pre-computed FPKM values were used as provided by the data depositor. For cross-dataset comparison, direction of change (log2FC sign) was compared between the two datasets at matching or nearest time points. No batch correction was applied, as the datasets were analyzed separately on their respective platforms (HiSeq vs. HiSeq X Ten). Protocol differences in ischemia duration (45 vs. 30 min) and sequencing platform are acknowledged as limitations.
Gene Set Enrichment Analysis (GSEA)
GSEA was performed using the fgsea R package (v1.20) with genes ranked by signed -log10(P-value) x log2FoldChange. The Nrf2-ARE gene set was curated from established literature (Itoh et al., 1997; Hayes & Dinkova-Kostova, 2014; Dodson et al., 2019; Stockwell et al., 2017) comprising 47 genes (43 detected in the ranked gene list): classical Nrf2 target genes (Gclc, Gclm, Gsr, Txnrd1, Prdx1, Sod1, Sod2, Cat, Hmox1, Nqo1, Txn1, Srxn1), glutathione metabolism enzymes (Gsta1-4, Gstm1-5, Gstp1, Gstt1-2), ferroptosis core genes (Gpx4, Slc7a11, Fth1, Ftl1, Tfrc, Acsl4, Ptgs2), Nrf2 regulatory factors (Sqstm1, Keap1, Nfe2l2), and NADPH/redox enzymes (G6pdx, Pgd). The Ferroptosis gene set was obtained from FerrDb (Zhou & Bao, 2020). The Hallmark gene set collection from MSigDB was used for broad pathway annotation. Permutation testing (10,000 permutations) was used to assess significance.
Single-Nucleus RNA-seq Analysis
The snRNA-seq dataset GSE240848 was processed using Cell Ranger (10x Genomics) and downstream analysis was performed in Seurat v5 / Scanpy v1.9. Quality control removed nuclei with >10% mitochondrial reads, <200 or >5,000 detected genes. After QC, 48,633 nuclei were retained. Normalization was performed using SCTransform. Dimensionality reduction was performed by PCA followed by UMAP. Cell types were annotated using canonical marker genes: Cardiomyocyte (Tnnt2, Myh7, Actc1), Fibroblast (Col1a1, Col1a2, Dcn), Endothelial (Pecam1, Cdh5, Kdr), Macrophage (Cd68, Csf1r, Adgre1), Neutrophil (S100a8, S100a9), T Cell (Cd3e, Cd3d), B Cell (Cd19, Cd79a), Pericyte (Pdgfrb, Rgs5), and Smooth Muscle Cell (Acta2, Myh11).
Cell-Cell Communication Inference
Cell-cell communication was inferred using two independent methods: CellChat (v1.6) and CellPhoneDB (v4, via the LIANA framework). Both methods were applied to the normalized snRNA-seq count matrix with the same cell type annotations. For CellChat, the default ligand-receptor database (CellChatDB.mouse) was used, and significant interactions were identified by permutation test (P < 0.05). For CellPhoneDB, significance was assessed using the built-in permutation framework. Consensus interactions were defined as ligand-receptor pairs identified as significant by both methods with matching sender and receiver cell types. Analysis focused on signaling from myeloid cells (Macrophage, Neutrophil) to Cardiomyocyte and Pericyte populations.
Proteomics Data Processing
Three independent proteomics datasets were analyzed: PXD035154 (TMT10plex isobaric labeling, mouse cardiac I/R, IsobarQuant quantification; Pool 1: 2-day I/R, Sham n = 2, I/R n = 3, ~6,400 gene products; Pool 2: 2-week I/R, Sham n = 2, I/R n = 3, ~6,500 gene products), PXD040135 (label-free quantification, mouse cardiac I/R 1 h LAD + 24 h reperfusion, n = 3 per group, 999 quantifiable proteins), and PXD046631 (label-free quantification, mouse cardiac I/R 30 min + 6 h reperfusion, n = 4 per group, 2,308 gene products with matched mRNA and protein data). MaxQuant output (proteinGroups.txt) was processed to remove contaminants and reverse hits. Protein intensities were log2-transformed. For cross-platform comparison, gene symbols were mapped to human orthologs via the mygene Python package using HomoloGene. Proteins detected in at least 2 of 3 replicates per group were retained for analysis.
mRNA-Protein Discordance Analysis (Residual Method)
For the PXD046631 dataset (2,308 genes with matched mRNA and protein measurements), mRNA-protein discordance was quantified by linear regression of log2(protein fold change) against log2(mRNA fold change). The regression yielded R = 0.253 (slope = 0.144), indicating weak genome-wide mRNA-protein correlation. For each gene, the Studentized residual from this regression was computed. Positive residuals indicate protein abundance exceeding mRNA-based predictions (suggesting translational activation or protein stabilization); negative residuals indicate lower-than-expected protein (suggesting translational repression or protein degradation). Directional concordance was defined as agreement in the sign of mRNA and protein log2FC (both positive or both negative).
OGT Substrate Enrichment Analysis
High-stringency human OGT protein interaction partners (n = 782) were obtained from OGT-PIN v2.0 (Ma et al., 2021; https://oglcnac.org/ogt-pin/), a curated database of experimentally identified OGT-interacting proteins. Mouse DEGs were mapped to human orthologs by case-insensitive matching (mouse gene symbols uppercased to human equivalents). Enrichment of OGT interactors among DEGs was assessed by Fisher's exact test (background: 20,000 human protein-coding genes). Functional categorization of network genes (O-GlcNAc Cycling, NRF2 Pathway, Ferroptosis Core, Antioxidant Targets, Signaling) was performed to assess pathway-specific overlap patterns.
External Validation Using GTEx and GSE57338
GTEx v8 median TPM values for Heart - Left Ventricle (n = 431 healthy donors) were queried via the GTEx Portal API v2 for core genes: OGT (ENSG00000147162.13), NFE2L2 (ENSG00000116044.15), HMOX1 (ENSG00000100292.16), SLC7A11 (ENSG00000151012.13), and BACH1 (ENSG00000156273.15). For GSE57338 (313 human left ventricular samples), Affymetrix probe-to-gene mapping was performed using the GPL11532 platform annotation. Differential expression between 177 HF and 136 NF samples was assessed by Welch's t-test. Directional consistency with the GSE193997 discovery DEGs was evaluated by binomial test (null hypothesis: 50% same direction). For cross-platform comparison, mouse gene symbols from GSE193997 were matched to human symbols (case-insensitive) on the GSE57338 array (5,514 common genes).
WGCNA and Network Analysis
Weighted gene co-expression network analysis (WGCNA) was performed on the GSE193997 time-series dataset (24 samples: Sham, Ischemia, and I/R at 0.5/1/1.5/3/6/12 h; n = 3 each). Genes with CPM > 1 in at least three samples were retained (14,774 genes), and the top 4,000 most variable genes were selected for signed network construction. The soft-thresholding power β = 3 was selected to achieve scale-free topology fit R² = 0.935. The topological overlap matrix (TOM) was computed, and hierarchical clustering (average linkage) of TOM-based dissimilarity (1 − TOM) was performed. Co-expression modules were identified by adaptive tree cutting (dynamic hybrid cut), with modules containing fewer than 15 genes reassigned to the grey (unassigned) module. Module eigengenes (first principal component of scaled expression) were correlated (Pearson r) with a binary I/R trait (Sham = 0, all I/R groups = 1). Gene significance (GS; |Pearson r| with I/R trait) and module membership (MM; |Pearson r| with module eigengene) were computed for all genes; hub genes were defined as those with MM > 0.8 and high GS. External validation of module eigengenes was performed by projecting GSE193997 module definitions onto the independent GSE307385 cohort (15 samples), with eigengenes computed via singular value decomposition (SVD) of z-score-normalized log2(FPKM) expression and compared by Welch's t-test.

Boolean network modeling of the 27-node O-GlcNAc-Nrf2-ferroptosis regulatory circuit, including attractor analysis, in silico perturbation, and robustness testing, is described in a companion paper focused on computational modeling and drug repurposing. The present study focuses on the multi-omics discovery and cell-type-resolved characterization of this regulatory axis.
Quantification and Statistical Analysis
All statistical analyses were performed in R (v4.2) and Python (v3.10). Key software packages and versions are listed in the Key Resources Table. Statistical details for each analysis are provided in the corresponding Method Details subsection and Figure Legends. Briefly:
Bulk RNA-seq differential expression: DESeq2 with Benjamini-Hochberg FDR correction; significance threshold: adjusted P < 0.05.
GSEA: fgsea with 10,000 permutations; significance threshold: FDR < 0.25 (standard GSEA threshold).
Cell-cell communication: CellChat permutation test (P < 0.05); CellPhoneDB built-in permutation (P < 0.05); consensus defined by overlap of both methods.
mRNA-protein residual analysis: linear regression (scipy.stats.linregress); Studentized residuals; concordance defined as same-sign log2FC.
OGT substrate enrichment: Fisher's exact test (one-sided), background = 20,000 human protein-coding genes.
Directional consistency (GSE57338): binomial test (one-sided) against null probability of 0.5.
Cross-platform comparisons used direction of change (log2FC sign) rather than absolute effect sizes, given differences in measurement scales between RNA-seq and microarray platforms.
For mouse bulk RNA-seq experiments: GSE193997, n = 3 per group (8 time points, 24 total); GSE307385, n = 3 per group (5 time points, 15 total). For proteomics: PXD046631, n = 4 per group; PXD040135, n = 3 per group; PXD035154, Sham n = 2, I/R n = 3 per pool. For the human validation cohort (GSE57338), n = 177 HF and 136 NF. For GTEx v8, n = 431 healthy left ventricular donors. Statistical tests for each analysis are described in the corresponding Method Details subsection.
Sample sizes (n) and statistical tests are reported in each Figure Legend. No statistical methods were used to predetermine sample size. As this study constitutes a secondary computational analysis of publicly available datasets, randomization and blinding were not applicable to the re-analysis; experimental details regarding randomization and blinding in the original studies are provided in their respective publications.
